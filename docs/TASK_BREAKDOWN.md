## 风险与缓解

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|---------|
| 招聘平台反爬升级导致插件失效 | 中 | 高 | 硬编码适配 + AI 兜底通用选择器；用户可手动粘贴 JD |
| LLM API 服务不稳定 | 低 | 中 | 规则解析做主力兜底；缓存避免重复调用 |
| Supabase 服务中断 | 低 | 中 | 本地 SQLite 缓存关键数据 |
| 用户隐私数据泄露 | 低 | 极高 | 分级存储 + RLS + LLM 代理过滤 + 不存密码 |
| 浏览器插件被 Chrome Web Store 拒绝 | 中 | 中 | 提供手动安装（开发者模式）方案 |
