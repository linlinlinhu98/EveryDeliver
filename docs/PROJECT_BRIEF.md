# EveryDeliver 项目简介

---

## 一句话版（<50字）

> **AI 驱动的求职效率引擎 — 上传简历，导入 JD，AI 精准优化，一站式投递追踪。**

---

## 1 分钟版（~180字）

> 求职者在不同平台搜寻匹配岗位、定制简历、追踪投递，每投一个岗位平均花费 2 小时。EveryDeliver 是一个 AI 驱动的半自动投递辅助平台。用户上传简历后，系统通过浏览器插件抓取或手动导入 JD，由 JD Parser（规则+LLM 三层解析）自动提取关键要求，AI 对简历进行 5 级风险分级的精准优化（三色高亮标记改动），用户确认后跳转平台手动投递。系统自动记录并生成三视图追踪看板，含 7 个状态管理和 8 类智能提醒。隐私三级分级存储（敏感字段本地加密，不送 LLM），成本约 ¥0.86/月/用户。桌面端 + 浏览器插件 + 微信小程序三端覆盖。

## 完整版（~900字）

### 一、项目背景与痛点

当前求职场景中，求职者面临两大核心痛点：

1. **效率低下**：需要在官网、公众号、BOSS 直聘、猎聘、小红书等多平台搜寻匹配岗位，每投递一个岗位，需要针对 JD 定制简历、准备面试，平均耗时 2 小时以上，且缺乏系统性的投递追踪手段。

2. **匹配度不透明**：求职者拿到感兴趣的 JD 后，不确定自身简历与岗位要求的具体差距，不知道哪些地方需要优化，如何优化才能提升通过初筛的命中率。

### 二、解决方案

EveryDeliver 是一个面向求职者的半自动投递辅助平台，核心定位是"通过 AI 智能筛选与适配提升投递效率"。平台覆盖用户使用的完整流程：

**用户画像与隐私保护**：支持简历上传（PDF/Word/TXT）和 Agent 对话式填写，采用三级隐私分级存储机制——敏感字段（手机号、身份证号、籍贯、完整生日）本地 AES 加密永不上传；通用字段脱敏后方可送 LLM 处理。

**职位获取与智能过滤**：支持浏览器插件抓取（BOSS 直聘 + 猎聘，借助用户登录态）和手动导入两种 JD 来源。系统自动执行三层过滤：黑名单命中直接丢弃、与历史投递记录重复则移入重复列表并提示原因、薪资低于偏好或城市不匹配则标记但不删除。黑名单支持 4 种自动检测机制（JD 关键词匹配 + 行业分类警告 + 工商信息查询 + 用户历史行为推断）。

**JD Parser（核心模块）**：采用"规则解析（主力）+ LLM 解析（兜底）+ 用户反馈（持续优化）"三层架构。针对 BOSS 直聘和猎聘硬编码 DOM 选择器提取结构化字段（薪资、地点、学历、经验年限、公司名），规则无法覆盖的语义字段（技能提取、职责总结）降级到 DeepSeek-V3 进行 LLM 解析。解析结果附带置信度评分，用户修正后系统缓存并定期提取为新的解析规则。JD 具备生命周期管理（active → expiring_soon → expired）。

**简历智能匹配与优化**：匹配度评估采用"硬性 60% + 软性 30% + 隐性 10%"公式，技能分为直接相关（权重 ×2.0）和通用技能（权重 ×0.5）。AI 简历优化支持询问/自动双模式，按改动风险分为 5 级：措辞优化（🟢自动应用）、侧重点调整（🟡自动应用）、GitHub 化用（🟠轻提示）、新增关键词（🔴弹窗确认）、新增经历（🔴强制确认）。改动以黄/橙/红三色高亮标记，每处改动附带原因说明、参考来源和面试准备建议。

**投递追踪看板**：三视图并存（表格 + 看板 + 时间线），7 个标准状态（待投递→已投递→HR已读→已沟通→面试邀请/已拒绝→已入职），8 类智能提醒（7/14天无响应、面试前24h/1h、面试后跟进、JD过期等）。以"公司名+标准化岗位名"作为防重复键，记录可删除但防重复键永久保留。

**简历模块化系统**：完整简历可自动拆解为 8 类模块（教育背景、实习经历、项目经历、技能证书、资质证书、获奖荣誉、语言能力、个人总结），每类模块独立存储和版本管理（50 版本上限），支持从模块库中选择拼接为新的完整简历。每类模块经过 5 维质检（陈旧/简略/缺量化/无标签/时间冲突）。投递时自动冻结简历快照。5 套简历模板（极简经典/应届清新/双栏商务/英文标准/创意视觉）。

**面试准备清单**：AI 自动生成技术题、行为面试题和精神风貌准备建议，与投递看板联动（面试邀请 → 自动加入准备清单）。简历新增内容也会自动生成对应的面试准备项。

**多端架构**：桌面端（Tauri v2 + React）为主战场、完整功能、独享编辑权；浏览器插件（Manifest V3）负责 JD 抓取和表单半自动填充（填充 → 用户确认 → 手动提交）；微信小程序为辅助入口（查看 + 记录 + 下载）。三端通过 Supabase Realtime 实现云端同步。

### 三、技术亮点

- **国产模型策略**：DeepSeek-V3（JD解析/匹配/说明）+ Qwen-Plus（简历优化/面试建议）+ Qwen-Max（面试题生成），成本仅 ¥0.86/月/用户，是 Claude API 的 1/15
- **隐私优先**：三级分级存储 + 所有用户数据表 RLS 强制隔离 + LLM 代理层敏感字段过滤 + 不存储任何第三方平台密码/Cookies
- **合规设计**：浏览器插件首次启用强制告知弹窗（工作方式 + 合规风险 + 隐私保护），用户自愿承担第三方平台协议风险
- **AI 深度整合**：不是 LLM 套壳，而是规则引擎 + LLM 兜底 + 用户反馈的三层持续优化架构
